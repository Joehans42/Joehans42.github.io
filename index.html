The Data

Introduction The data in this project consists of two separate data sets. The first data set is a list of Wikipedia articles. The Wikipedia articles can be different depending on the choosing of an initial starting page. In this data set however, the initial starting page is pre-selected as the Wikipedia page for Albert Einstein. The first data set is thus a network consisting of the articles that can be reached by clicking the links on the pages within 2 clicks.
